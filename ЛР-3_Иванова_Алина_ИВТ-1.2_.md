### Задание 1.1
### Постановка задачи: 
Создать некоторую структуру с указателем на некоторую функцию в качестве поля. Вызвать эту функцию через имя переменной этой структуры и поле указателя на функцию.

**Список идентификаторов:**

| ИМЯ | ТИП    | СМЫСЛ                              |
|-----|--------|------------------------------------|
|  MyStruct  |     |        |
|    |     |     |
|    |     |               |

**Код программы:**

```c
#include <stdio.h>

typedef void (*func_ptr)(void);
struct MyStruct {
    func_ptr function;
};

// функция, которую мы будем вызывать
void my_function() {
    printf("функция вызвана через указатель в структуре\n");
}

int main() {
    struct MyStruct my_instance;
    my_instance.function = my_function;
    my_instance.function();
    
    return 0;
}
```
